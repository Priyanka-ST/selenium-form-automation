# Selenium Web Form Automation Project

## Overview
This project automates a web form using Selenium WebDriver with Java and TestNG.

## Tools Used
- Java
- Selenium WebDriver
- TestNG
- Maven
- Eclipse IDE

## Application Tested
https://www.techlistic.com/p/selenium-practice-form.html

## What This Project Covers
- Browser setup and teardown
- Explicit waits for dynamic elements
- Form input automation
- Radio button and date selection
